First-time setup — Google Cloud credentials (takes 3 minutes)

Sign in with naturedispatch@gmail.com for all steps below:

1. Go to console.cloud.google.com and create a new project (e.g., "ND Load Planner")

2. Go to APIs & Services → Library, search for Gmail API and Enable it

3. Go to APIs & Services → Credentials → click + Create Credentials → OAuth client ID

4. If prompted, configure the OAuth consent screen: choose External, fill in app name, use naturedispatch@gmail.com as the contact email, and save. Add naturedispatch@gmail.com as a Test user.

5. Back in Credentials → Create OAuth client ID: Application type = Web application

6. Under Authorized JavaScript origins, add the URL where you open this app (e.g., http://localhost or your domain). If opening as a file, add http://localhost.

7. Copy the Client ID and paste it below:
